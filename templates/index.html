Maths Server

This is a simple maths server that can perform basic operations like addition, subtraction, multiplication, division, and exponentiation.

Endpoints

  • /: This is the home page. It lists all the available endpoints.
  • /expression: This endpoint evaluates the given expression. The expression can contain numbers, operators, and functions.
  • /history: This endpoint returns the history of all the calculations that have been performed.

Examples

To add 2 and 3, you can use the following endpoint:

USE= /2/plus/3
answer = 5

To subtract 5 from 10, you can use the following endpoint:

USE= /10/minus/5
answer = 5

To multiply 2 and 3, you can use the following endpoint:

USE= /2/into/3
answer = 6

To divide 10 by 2, you can use the following endpoint:

USE= /10/dividedby/2
answer = 5

To raise 2 to the power of 3, you can use the following endpoint:

USE= /2/power/3
answer = 8

To view previous 20 calculations:

USE= /history
{question = 2+3
answer = 5}
{question = 10-5
answer = 5}
{question = 2*3
answer = 6}
{question = 10/2
answer = 5}
{question = 2**3
answer = 8}
